모집단 : 실험이나 조사 등을 수행할 때 계측 또는 관측 대상인 사람이나 사물의 집합
개체 : 모집단에 포함되는 요소
전수 조사 : 모집단의 모든 개체를 대상으로 하는 조사 ex) 국세 조사
	1) 표본 조사 : 대부분 모집단의 일부를 추출하여 조사(전수 조사는 금전적, 시간적으로 비용이 많이 들기 때문)
	2) 표본 : 표본 조사에서 추출한 개체의 집합
	3) 표본의 크기 : 표본에 포함되는 개체 수

데이터 해석 : 표본의 정보를 이용하여 모집단의 밝혀지지 않은 성질에 대해 어떠한 결론을 이끌어 내는 것
통계학 : 그 방법적 기초를 제공하는 학문
	1) 개념의 계량화
	2) 예측 
	3) 가설 검정
	4) 분류

데이터 : 측정값의 집합
변량, 변수 : 계측 대상이 되는 속성(키)
1차원 데이터(단변량 데이터) : 한 가지 변량만 계측한 데이터
2차원 데이터 : 두 가지 변량 계측한 데이터

양적 변량 : 관측값이 수치로 나타나는 변량
	1) 이산형 : 띄엄띄엄 떨어진 값밖에 얻을 수 없는 변량이나 데이터
		ex) 교통사고 건수, 병에 걸린 환자 수 등 계수(사람 수나 건수, 횟수, 개수 등)
	2) 연속형 : 연속된 값을 얻을 수 있는 변량이나 데이터
		ex) 키, 몸무게
양적 데이터 : 양적 변량을 계측한 데이터

질적 변량 : 속성이나 항목, 카테고리, 형태를 띠는 변량
질적 데이터 : 질적 변량을 계측한 데이터

횡단면 데이터 : 복수의 개체를 어느한 시점에서 본 관측값으로 이루어진 데이터
시계열 데이터 : 하나의 개체를 복수의 시점에서 본 관측값으로 이루어진 데이터
