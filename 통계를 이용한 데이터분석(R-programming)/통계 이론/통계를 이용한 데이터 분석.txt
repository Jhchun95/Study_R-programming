ㄱ. 기술 통계(Descriptive statistics)

*의미 
1. 영문 스펠링을 보면 '서술(Descriptive)하는 통계'라는 뜻이고, 데이터의 특성을 이해하기 쉽게 기술하는 통계
2. 측정이나 실험을 통해 수집한 데이터의 정리, 그래프나 숫자 등으로 요약, 표현하는 등 데이터의 특성을 규명하는 통계적 방법
- 예시 : 평균, 중앙값, 최빈값, 범위, 분산, 표준편차, 사분위수, 도수분포표, 왜도, 첨도
- 그래프 : 파이 그래프(Pie char), 상자그림(Box plot), 막대 그래프(Bar plot), 히스토그램(Histogram)

*정리
탐색적 데이터 분석과 기술 통계의 차이점
1. 분석 목적이 다르다
- 기술 통계 : 분석 결과를 해석하는 것보다 데이터를 설명, 즉 단순히 기술하는 것이 목적
- 탐색적 데이터 분석 : 가설 수립, 변수 간의 관계 파악, 트랜드 파악


ㄴ. 추론 통계

*의미
1. 모든 대상의 데이터 수집이 불가능하거나, 소수의 대상을 가지고 연구하는 것이 경제적이고 효율적일 때 
모집단에서 추출한 표본 데이터를 가지고 그 모집단의 특성 및 가능성 등을 추론해내는 통계적인 방법

*정리
1. 추론 통계는 바탕인 기술 통계량이 있어야 한다.
2. 일반적인 추론은 표본의 통계 결과로 모집단의 특성을 추론 및 검증하거나 또는 다른 집단 간의 비교 등에서 차이점이 유의한지를 검증


귀무가설과 대립가설
1. 귀무가설(기호 H0) : 차이가 없거나 의미있는 차이가 없는 경우의 가설
2. 대립가설(기호 H1) : 연구자가 연구를 통해 입증되기를 기대하는 예상이나 주장하는 내용
*귀무가설이 채택되면 대립가설은 기각된다. 귀무가설이 기각되면 대립가설이 채택된다.

유의수준과 유의확률
1. 유의수준 : 가설 검정에서 가설 채택 여부를 결정하는 기준값
	일반적으로 α로 표시하고 95%의 신뢰도를 기준으로 한다면 (1-0.95)인 0.05 값이 유의수준 값이 된다.
2. 유의확률(p-value) : 집단 간의 차이가 우연히 일어났을 확률
	우연히 집단 간의 차이가 발생했을 확률인 유의확률이 유의수준보다 크면 '차이가 없다'는 가설인 귀무가설이 채택된다.
* p-value >= 유의수준(예를 들어, 0.05) : 귀무가설 채택, 대립가설 기각
* p-value < 유의수준(예를 들어, 0.05) : 귀무가설 기각, 대립가설 채택

점추정과 구간추정
1. 점추정 : 모수가 특정한 값일 것이라고 추정, 미지의 분포에 대하여 가장 근사한 단일 값을 구함
2. 구간추정 : 모수가 특정한 구간 안에 있을 것이라고 추정
	예를 들어, '모집단의 평균 점수는 95% 확률로 70~80 사이라고 말할 수 있다'와 같은 추정
	* 모집단 : 정보를 얻고자 하는 관심 대상의 전체 집합
	* 모수 : 모집단의 특성을 수치로 나타낸 것
	* 통계량 : 표본의 특성을 수치로 나타낸 것
